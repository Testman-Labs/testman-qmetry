## Configure Repositorio

1. Create a folder called settings inside .github in your test case management repository
   
2. Copy the settings.json file from test/loader_settings to the .github/settings directory

3. Configure your jirabase_url and test case custom fields in Qmetry

</br>  

**Important:** Replace vars 'HERE_YOUR_SECRET_KEY' in workflows
